Imagens

Diferenças entre JPEG, GIF e PNG

O que é JPEG

Imagem em JPEG de uma lagoa ao entardecer, com árvores ao fundo e um deck sobre a água. É o formato mais utilizado na hora de salvar uma imagem, suportando cerca de 16,8 milhões de cores, o formato foi criado em 1983. O JPEG (Joint Photografic Experts Group) proporciona tamanhos de imagens menores, sendo interessante na hora de salvar a imagem, porém, a qualidade fica menor, pois é utilizado um processo de compressão, em que é eliminado alguns dados irrelevantes do arquivo.

Para não ter tanta perda de qualidade, alguns programas de edição de imagem permitem que o usuário escolha o nível de compressão, quanto menor esse nível, melhor será a qualidade da imagem.

O que é GIF

GIF de um gato preto e branco sentado em frente a uma pequena mesa de madeira, mexendo em um notebook e um mouse branco. O formato GIF (Graphics Interchange Format) é bastante conhecido, principalmente nas redes sociais, por proporcionar uma breve animação. Nos formatos citados nesse post é o que menos suporta cores, cerca de 256.

Uma das vantagens é que o GIF permite fundo transparente, assim, nas áreas que está transparente, a imagem ficará no fundo da página em que ela está inserida. Além disso, utiliza tamanhos de arquivos reduzidos, assim como o JPEG.

O que é PNG

Imagem em PNG de uma plantação com montanhas ao fundo, ceu azul e nuvens brancas. O PNG possui milhares de cores disponíveis, é um formato bastante utilizado devido a qualidade de imagem que apresenta, porém não é suportado em todas as plataformas.

O PNG (Portable Network Graphics) fornece os recursos do GIF, como é o caso da transparência do fundo da imagem.

O que é SVG

Atualmente, há um aumento no uso do formato SVG (Scalable Vector Graphics), uma vez que proporciona alta qualidade para a imagem, sem precisar ter um tamanho de arquivo grande e permite a transparência no fundo da imagem. Além de se adaptar bem no uso de Smartphones e tablets.

O tamanho da imagem também é muito importante pois influencia na velocidade do seu site. Um tamnho ideal seria de até 1500, mas isso vai depender onde a imagem sera inserida, é logo, vai para o conteudo, ficará no rodapé? gimp é uma ótima plataforma gratuita que oferece recursos de edição de imagem.

Semântica no HTML

Semântica é o estudo do significado e interpretação de palavras. Aplicando isso ao HTML, significa a melhor compreensão do HTML para o computador. Antes do HTML5, utilizava-se somente divs para dividir conteúdos e classes para estilizar, sem contar com a compreensão do próprio computador para “entender” o HTML.

Os principais elementos HTML semânticos para layout da página são:

  • article - tag especifica conteúdo independente e autocontido.
  • aside - tag indica conteúdos laterais, geralmente utilizada para menus laterais.
  • footer tag é o rodapé da pagina.
  • header - é um elemento HTML que representa o cabeçalho de uma página web.
  • main - é a tag onde colocamos o conteúdo principal da página. Para melhor uso, só devemos ter um main por página, onde iremos apontar o conteúdo de maior relevância, sendo textos, imagens, links, etc.
  • nav - usamos para menus interativos, como por exemplo menus que encaminham a página de “sobre” ou “produtos”. O
    pode ser usado em qualquer momento na página, onde tenhamos agrupamento de links.
  • section - que define seções de conteúdos — seja dentro de um texto ou conteúdo principal, conseguimos dividir melhor esse conteúdo, deixando-o melhor organizado.

    • É usar a tag address de endereço para que o navegador entenda de que se trata de um endereço e pode até indicar o maps para o usuário.
